| 5 | Առաջին code review-ի ստացում | Հետադարձ կապի ռիթմ | 🦴 ՄԵՏԱ §4 |

📊 §W.2 · 30-60-90 Performance Reviews (L12 Բացի Փակում)

Ամեն փուլի վերջում՝ կառուցվածքային ինքնագնահատում + հետադարձ կապ:

### 📋 30-օրյա Review — «Ինչպե՞ս եմ հարմարվում»

| Կատեգորիա | Հարց ինքդ քեզ | Իդեալական Պատասխան |
|------------|---------------|-------------------|
| 🛠️ Տեխնիկական | Կարողանո՞ւմ եմ կոդ գրել առանց օգնության | «Այո, փոքր խնդիրները ինքնուրույն» |
| 🤝 Թիմային | Ինչպե՞ս եմ հաղորդակցվում | «Հարցնում եմ, երբ չգիտեմ; կիսվում եմ, երբ գիտեմ» |
| 🎯 Արդյունք | Ի՞նչ եմ ավարտել | «1 փոքր feature, 2 bug fixes, 3 code reviews» |
| 🧭 Ուղղություն | Ի՞նչ եմ սովորել | «Company workflow, code style, deployment process» |

### 📋 60-օրյա Review — «Ինչպե՞ս եմ ներդրում անում»
### 📋 90-օրյա Review — «Ինչպե՞ս եմ աճում»

📝 Review-ի Ձևաչափ:
1. Պատրաստիր 1-էջանոց self-assessment (կետ առ կետ)
2. Խնդրիր 15-20 րոպե manager-ի հետ
3. Հարցրու. «Ի՞նչ եմ անում լավ, ի՞նչ կարող եմ բարելավել»
4. Գրիր արդյունքները և հաջորդ քայլերը
5. Կիսվիր թիմի հետ (ըստ անհրաժեշտության)

💡 Review-ը ոչ թե «գնահատում» է, այլ «աճի քարտեզ»:
