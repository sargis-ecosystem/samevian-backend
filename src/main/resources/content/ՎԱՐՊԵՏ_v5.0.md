| Մազերը կտրված | Խնամք = շարունակականություն = հուսալիություն |

| Սիրելի օծանելիք | Թեթև ներկայություն, ոչ ճնշող |

**🎵 Երաժշտության Թույլտվության Արձանագրություն**

### A.3 · Փուլ 3 · Վստահության Կառուցում (Trust Building)

**🎒 2-Հավաքածուի Պահուստավորման Համակարգ (2-Kit Redundancy System)**
2 զամբյուղ ամեն գործիքից: 1-ը տանը պահեստ: Եթե մեկը փչանում է — աշխատանքի ժամկետը ԱՆԽԱԽՏ է մնում:

> *💡 Սա **բիզնեսի շարունակականության պլանավորում է (business continuity planning)** — ավագ մակարդակի քայլ, ոչ պատահական:*

**💎 3-Մակարդակի Գործիքների Տնտեսում (3-Tier Tool Economy)**

| Մակարդակ | Օրինակներ | Ով է գնում |
|----------|----------|-----------|
| Պրոֆեսիոնալ | Լազեր, սալիկ կտրող, որակյալ մալաներ | ԵՍ — ներդրում որակի մեջ |
| Օժանդակ | Մանր գործիքներ, երկրորդական իրեր | ԵՍ — բավարար լավը |
| Սպառվող | Դույլ, սվեռլո, շպատել, նյութեր | ՏԵՐԸ — վճարվում է նախագծից |

**🤝 Ձևականության Վերացում (Formality Dissolution) — Անցման Քայլ**
*«Դեռ երկար եմ աշխատելու ձեր տանը · անցնենք դու-ից դու-ի»*

Առանցքային քայլ — կարծես հաճախորդի հարմարության համար, իրականում ստեղծում ես **հանգիստ աշխատանքային միջավայր** 2-3 շաբաթվա համար: Բայց **հաճախորդն է ընտրում**, ոչ թե պարտադրվում:

> *🛡️ «Ես չեմ հետապնդում այլ մտքեր» — հստակ սահմանի հաստատում: Մասնագիտական դիրքավորում:*

**❤️‍🩹 Հոգեբանական Դեր (Psychological Role)**
*Ես վերցնում եմ հոգեբանի կամ լավ ընկերոջ դերը իմ վրա*

Հաճախորդը լարված է — մեծ ծախս, փոշի, սթրես: Դու **գիտակցաբար** վերցնում ես զգացմունքային բեռը քո վրա: Սա արհեստ չէ: Սա **հոգատարություն (care)** է:

**☕ 45/15 Ռիթմ — Pomodoro Արձանագրություն**

**💬 Զրույցի Ճարտարագիտություն (Conversation Engineering) — 3 Կանոն**

**✅ ԹՈՒՅԼԱՏՐԵԼԻ Թեմաներ**

> • Տան իրեր + պատմություն

> • Ազգային պատկանելիություն

**📈 Հեղինակության Ճարտարագիտություն (Reputation Engineering) — 5 Ցանկալի Հատկանիշ**
Գիտակցված ճարտարագիտություն — ոչ պատահական:

### A.4 · Փուլ 4 · Աշխատանքի Մեկնարկ + Ինտելեկտ (Work Start + Intelligence)

**🎯 Օր 1-2 · Քանդում և Փոշի**
> • Ամբողջ քանդումը front-loaded (առաջնային)
> • Աղբը մեշոկներով դասավորված հատկացված վայրում
> • Գրունտ բոլոր մակերեսներին — փոշու աղբյուրի վերացում
> • Մաքրում օրվա վերջում — տեսանելի առաջընթաց հաճախորդի համար

**💰 Փոքր Վճարման Ստուգում (The Small Payment Probe)**
Առաջին հանգրվանից (milestone) հետո` փոքր գումար խնդրելը: Մակերեսային պատճառ — առօրյա ծախս: Իրականում` **4-շերտանի թեստ**:

| Շերտ | Ինչ ես չափում |
|------|---------------|
| 1. Վճարման հոգեբանություն | Հեշտ է տալիս? Սթրեսվա՞ծ է: |
| 2. Վստահության հաստատում | Պարտավորվում է գործողությամբ, ոչ միայն խոսքերով |
| 3. Դրամական հոսքի ձևավորում | Ավելի հաճախ վճարումներ պե՞տք են, թե՞ քիչ: |
| 4. Բավարարվածության ստուգում | Դիմադրություն կա՞ = դժգոհություն կա |

**🧠 Հաճախորդների Սեգմենտավորման Շրջանակ (Client Segmentation Framework)**

| Մակարդակ | Ազդանշան | Ներգրավում |
|----------|---------|-----------|
| 🚫 Էժան փնտրող | Գնով տարված | Նախապես զտված |
| 🚫 Ժլատ | Դիմադրություն ամեն գումարի | Նախապես զտված |
| ✅ Վճարունակ + համարժեք | Առողջ հոգեբանություն | Հիմնական շուկա |
| ✅ Վստահելի/Երաշխավորված | Նախապես վաճառված | Պրեմիում վերաբերմունք |

**📜 Հեղինակության Պատերազմ (Reputation Warfare) — Reviews-ի Ռազմավարություն**
10-ից 1 հաճախորդը փորձել է ցեխ շպրտել հեղինակության վրա: Դու **հանրությանն** ես ուղղում պատասխանը, ոչ թե բողոքողին:

*Ամեն review արյունով և քրտինքով*

**◆ Շարունակական Հաճախորդի Գնահատում (Continuous Client Assessment)**
Նախնական 5-րոպեանոց ախտորոշումը մեկանգամյա չէ: Ամենօրյա ճշգրտում.
> • Տարածական մտածողություն — կարո՞ղ է պատկերացնել տարածությունը:
> • Տրամաբանության ոճ — հաջորդական, քաոսային, պահային:
> • Ճանաչողական հասանելիություն — ուղեղը զբաղվա՞ծ է, թե՞ ազատ:
> • Հաղորդակցման նախընտրություն — դիագրամներ? Ցուցակներ? Ամփոփում?

### A.5 · Փուլ 5 · Տեղադրում + Ստորագրություն (Installation + Signature)

**🔧 Օր 3 · Ենթակառուցվածք (Infrastructure)**
> • Ջրի և էլեկտրականության մոնտաժ — ստանդարտ բարձրություններ
> • Հարմարեցում հաճախորդի հատուկ կարիքներին, եթե պահանջվում է
> • Բացատրել նպատակը աշխատելիս — կրթություն գործողության մեջ

**📐 Սալիկի Ալգորիթմ (Tile Algorithm)**
```
ԱԼԳՈՐԻԹՄ. Սալիկի Դասավորություն
ՆԱԽՊԱՅՄԱՆՆԵՐ. - Պատերը պատրաստ են, հիդրոն չոր է - Համաչափության կենտրոնը գտնված է
ԱՆՓՈՓՈԽՆԵՐ. - Ամենափոքր սալիկը ≥ ամբողջ լայնության 50%-ը - Ոչ մի տգեղ փոքր կտոր պատերին
ԿԱՏԱՐՈՒՄ. 1. Սկսել 2-րդ շարքից (ոչ 1-ին) 2. Հատակի տեղադրումը ՎԵՐՋԻՆ 3. Առաջնահերթ գոտիներ. տեսանելի դռնից 4. 100% սոսինձ թե սալիկին, թե պատին 5. Մաքրել աշխատելիս
ՈՐԱԿԻ ԴԱՐՊԱՍՆԵՐ. - Ամեն շարք. ստուգել հավասարեցումը - Ամեն սալիկ. ստուգել սոսնձի ծածկույթը - Օրվա վերջ. ամբողջական մաքրում
```

public static void main(String[] args) {
    Master sargis = new Master();
    while (sargis.isWorking()) {
        sargis.restForDays(3);
        List<Lead> leads = sargis.generateLeads();
        List<Meeting> meetings = sargis.scheduleMeetings(leads);
        Project project = null;
        for (Meeting m : meetings) {
            Client client = sargis.diagnose(m, 5_MINUTES);
            if (sargis.isWorthWorkingWith(client)) {
                project = sargis.negotiate(client);
                break;
            }
        }
        sargis.execute(project);
        sargis.askForFeedback();
        sargis.updateReputation();
    }
}
```

⚖️ ՄԱՍ C · 10 Տրանսֆերաբիլ Սկզբունք + 2 DNA Ընդլայնում

**1. 🎯 Ժամանակի Արդյունավետություն (Time Efficiency Over Everything)**
Ոչ որակը, ոչ գինը — ժամանակի կառավարումն է #1: Զուգահեռ աշխատանք, սպասման նվազեցում, scope discipline:
> *Java: Async գործողություններ, զուգահեռ հոսքեր, deadline-ներով աշխատող սպրինտներ:*

**2. 🛡️ Շրջանակի Կարգապահություն (Scope Discipline)**
«Ինչ չեմ ես անում» — հստակ, նախապես: Scope creep = մահ:
> *Java: Interface պայմանագրեր, սահմանափակված խնդիրների սահմանումներ, «ոչ» ասել feature creep-ին:*

**3. 🎭 Իշխանության Փոխադարձում (Authority Reversal When Needed)**
Դու փորձագետն ես: Հաճախորդի հարգանքը ստանում ես հստակությամբ, ոչ ստրկամտությամբ:
> *Java: Ճարտարապետական որոշումներ — հակադարձել վատ պահանջներին: Խորհուրդ տալ, ոչ թե կուրորեն կատարել:*

**4. 🔬 5-Րոպեանոց Ախտորոշման Օրինաչափություն (5-Minute Diagnostic Pattern)**
10-ամյա օրինաչափությունների ճանաչում ↔ Senior Engineer-ի համակարգի նախագծման գնահատում:
> *Java: Code review — 5 րոպե կարմիր դրոշները նկատելու համար, ինչպես 5 րոպե դժվար հաճախորդին նկատելու համար:*

**5. 💰 Արժեքային Գնագոյացում (Value-Based Pricing, Not Time-Based)**
Գինը փորձառության դիմաց է, ոչ թե ժամերի: Հաճախորդների սեգմենտավորում գնագոյացման միջոցով:
> *Java: Դրույքաչափի սահմանում — դիրքավորիր քեզ որպես մասնագետ, ոչ թե ապրանք: Ընտրիր քո հաճախորդի մակարդակը:*

**6. 🧱 Ամենուրեք Պահուստավորում (Redundancy Everywhere)**
2 գործիքակազմ: Պահուստային պլաններ: Անխափան աշխատանքային ընթացք:
> *Java: Կրկնօրինակումներ, failover սերվերներ, սխալների մշակում, հետադարձի ընթացակարգեր:*

**7. 🤝 Վստահություն Հստակ Սահմաններով (Trust via Explicit Boundaries)**
«Ես չեմ հետապնդում այլ մտքեր» — հստակությունը ստեղծում է վստահություն, ոչ թե առեղծված:
> *Java: NDA-ներ, հստակ դերերի տարանջատում, հստակ հաղորդակցման քաղաքականություն:*

**8. 📜 Հեղինակությունը որպես Երկարաժամկետ Ակտիվ (Reputation as Long-Term Asset)**
100+ reviews = կուտակված վստահության կապիտալ, որը նվազեցնում է ապագա ձեռքբերման արժեքը:
> *Java: GitHub contributions, open source, տեխնիկական բլոգ, կոնֆերանսների ելույթներ:*

**9. 🔄 Շարունակական Բարելավում Հստակ Հետադարձ Կապով (Continuous Improvement via Explicit Feedback)**
«Կոնստրուկտիվ քննադատություն, ոչ գովաբանում» — կառուցվածքային, ժամանակով սահմանափակված հետադարձ կապի ծիսակատարություն:
> *Java: Code reviews, retrospectives, post-mortems — հետադարձ կապին վերաբերվել որպես նվերի:*

**10. 💎 Արհեստը որպես Ինքնության Միջոց (Craft as Medium for Identity)**
