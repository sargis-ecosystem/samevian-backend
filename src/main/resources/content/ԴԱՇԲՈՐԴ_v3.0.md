📈 §D.2 · Weekly SPC Charts (Շաբաթական Վիճակագրական Գործընթացի Վերահսկման Չարթեր) (Q2 Բացի Փակում)

Statistical Process Control — արտադրական որակի կառավարման մեթոդը կիրառված անձնական աճի վրա:

📊 Շաբաթական Չարթեր (5 հիմնական):

#### 1. Focused Hours Rolling Average (7-օրյա)
```
Շաբաթ 1: [▂▃▅▆▇] 2.1 ժ միջին
Շաբաթ 2: [▃▅▆▇█] 2.4 ժ միջին
Շաբաթ 3: [▅▆▇█▇] 2.3 ժ միջին
```
· Վերին վերահսկման սահման (UCL): միջին + 2σ
· Ստորին վերահսկման սահման (LCL): միջին - 2σ
· Եթե արժեքը դուրս է սահմաններից → հետաքննել պատճառը

#### 2. Commit Consistency Chart
```
Օր: Երկ Չոր Ուրբ Շաբ Կիր Երք Հնգ
Շ1: ● ● ● ● ○ ● ● → 6/7 ✅
Շ2: ● ○ ● ● ● ● ● → 6/7 ✅
Շ3: ● ● ○ ○ ● ● ● → 5/7 ⚠️
```
· Նպատակ` ≥5/7 օր ակտիվություն
· Եթե <4/7 երկու շաբաթ անընդմեջ → առաջարկել Failure Recovery

#### 3. Energy Variability Monitor
```
Էներգիայի տատանում (1-10 սանդղակ):
Շաբաթ 1: 6→7→5→8→6→7→6 → σ=0.9 (կայուն)
Շաբաթ 2: 8→4→3→6→5→7→8 → σ=1.8 (անկայուն) ⚠️
```
· Բարձր σ = անկանխատեսելի էներգիա → պլանավորման դժվարություն
· Առաջարկ` Body Setup-ի կիրառում (§A.2.5)

#### 4. Burn-Down Chart 🆕 (Agile)
```
Tier 1 Progress (5 դաս, 3 շաբաթ):
Week 1: ████████░░ 80% remaining
Week 2: █████░░░░░ 50% remaining
Week 3: ░░░░░░░░░░ 0% remaining ✅
```
· Իրական մնացորդ vs Իդեալական մնացորդ
· Deviation >20% → վերանայել Takt Time-ը

#### 5. Burn-Up Chart + Velocity Tracking 🆕 (Agile)
```
Completed Lessons by Week:
Week 1: ██ 2 lessons → Velocity: 2
Week 2: ███ 3 lessons → Velocity: 3
Week 3: █████ 5 lessons → Velocity: 2.5 avg
```
· Scope-ի ցանկացած ավելացում արտացոլվում է scope line-ում
· Velocity-ն օգտագործվում է ապագա Tier-ների ավարտը կանխատեսելու համար

📝 Շաբաթական Review Protocol (Վերանայման Արձանագրություն):
1. Կիրակի երեկոյան` 10 րոպե նվիրել չարթերի վերլուծությանը
2. Հարցնել ինքդ քեզ. «Ի՞նչն է ազդել այս շաբաթվա տվյալների վրա»
3. Գրանցել 1 actionable insight (օր. «Երեքշաբթի երեկոյան չսովորել»)
4. Պլանավորել հաջորդ շաբաթվա 1 փոքր բարելավում
